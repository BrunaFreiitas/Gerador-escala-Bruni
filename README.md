# Gerador de Escala 12x36

Ferramenta simples em Python para geração de escalas de trabalho no modelo 12x36.

## Motivação
Após 11 anos trabalhando na área da saúde, percebi como a organização de escalas pode ser complexa.

Este projeto foi criado durante minha transição de carreira para tecnologia.

## Funcionalidades
- Cadastro de profissionais
- Geração automática de plantões
- Simulação de escala

## Tecnologias
Python
